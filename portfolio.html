Personal Portfolio Website

A clean, responsive portfolio website built from scratch using HTML and CSS to showcase my projects and skills.

HTML CSS Responsive Design Web Design

Problem

As a Computer Science student, I needed a professional online presence to showcase my skills, projects, and experience to potential employers and collaborators. The portfolio needed to be simple, fast, and easy to maintain.

Solution

Built a static portfolio website using HTML and CSS with a focus on clean design, readability, and usability. Implemented responsive design principles to ensure the site works well on different screen sizes. Used semantic HTML for better accessibility and SEO.

Results

  • Professional online presence for career opportunities
  • Clean, accessible design with semantic HTML
  • Fast loading times with no dependencies
  • Easy to update and maintain

Raspberry Pi Greenhouse Monitoring System

A team project that uses Raspberry Pi and environmental sensors to monitor greenhouse conditions in real time.

Raspberry Pi Python Sensors Internet of Things Team Collaboration

Problem

Greenhouse environments require careful monitoring of temperature, humidity, and other conditions to ensure optimal plant growth. Manual monitoring is time-consuming and inefficient, making it difficult to respond quickly to environmental changes.

Solution

Developed an IoT monitoring system using Raspberry Pi connected to temperature and humidity sensors. Created a dashboard to display real-time sensor data and track environmental conditions across different greenhouse zones.

Results

  • Real-time monitoring of greenhouse conditions
  • Automated data collection reducing manual work
  • Visual dashboard for easy condition tracking
  • Collaborative project experience with hardware and software integration
